Class MenuItem

Description
Description | Vars (details) | Methods (details)

Κάθε αντικείμενο αυτής της κλάσης περιέχει τις πληροφορίες που χρειάζεται για να παρασταθεί στο menu. ΔΕΝ περιέχει πληροφορίες για τη θέση, η επιλογή της θέσης γίνεται μέσα από τη μέθοδο generateMenu της κλάσης PageComposer.
