Class ScheduleEntryLab

Description
Description | Vars (details) | Methods (details)

Δομή που περιέχει πληροφορίες ένα εργαστήριο. Χρησιμοποιείται στην παραγωγή πινάκων και λιστών που έχουν να κάνουν με το ωρολόγιο πρόγραμμα και καθορίζει παραμέτρους όπως τον τρέχων και μέγιστο αριθμό φοιτητών του εργαστηρίου.
