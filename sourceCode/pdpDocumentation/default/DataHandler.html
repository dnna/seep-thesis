		getLabs (line 101)
	
 
	
	

Επιστρέφει ένα δισδιάστατο πίνακα με εργαστηριακά τμήματα και πληροφορίες  για αυτά, φιλτραρισμένα με βάση διάφορες παραμέτρους, οι οποίες μπορούν  να συνδυαστούν. Οι πληροφορίες που περιέχονται αφορούν τη θέση στο  ωρολόγιο πρόγραμμα, τον τύπο, το εξάμηνο στο οποίο διδάσκονται κ.τ.λ.


και είναι οι εξής:  
  1. NAME -> Όνομα του εργαστηριακού τμήματος.
  2. DAY -> Το ID της ημέρας στην οποία διδάσκεται.
  3. dayName -> Το όνομα της ημέρας στην οποία διδάσκεται.
  4. tfrom -> Η ώρα έναρξης του εργαστηρίου.
  5. tto -> Η ώρα λήξης του εργαστηρίου.
  6. courseType -> Ο τύπος του μαθήματος (συνήθως "Εργαστήριο").
  7. labID -> Ο μοναδικός κωδικός του εργαστηρίακού τμήματος.
  8. courseID -> Ο μοναδικός κωδικός του μαθήματος στο οποίο ανήκει.
  9. courseTaughtSemester -> Το εξάμηνο στο οποίο διδάσκεται.

	
    
				
  • return: Δισδιάστατος πίνακας με εργαστηριακά τμήματα και  πληροφορίες που τα αφορούν. Η μια διάσταση είναι απλά ένα αριθμτικό  index, ενώ η άλλη είναι associative array με τις διάφορες πληροφορίες. Η  θέση με δείκτη numStudents περιέχει τον αριθμό των φοιτητών που έχουν  κληρωθεί σε αυτό το εργαστήριο, ή έχουν γραφτεί μόνιμα με κάποιον τρόπο.
