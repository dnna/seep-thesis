		hasAccess (line 168)
	
 
	
	

Ελέγχει αν ο χρήστης έχει πρόσβαση σε κάποιο module. Αυτό γίνεται


ελέγχοντας το αρχείο ModuleDescr.php του αντίστοιχου module και  συγκρίνοντας του ρόλους του χρήστη μέχρι να βρέθει κάποιος κοινός με  έναν από αυτούς του module. Αν στο module υπάρχει σαν ρόλος το anonymous  τότε αυτό σημαίνει αυτόματα ότι όλοι έχουν πρόσβαση, ακόμα και αυτοί  που δεν έχουν αυθεντικοποιηθεί. Αν στο module υπάρχει σαν ρόλος το  authenticated αυτό σημαίνει αυτόματα ότι όλοι οι αυθεντικοποιημένοι  χρήστες έχουν πρόσβαση, ανεξαρτήτως ρόλου.

	
    
				
  • return: Επιστρέφει true αν έχει πρόσβαση ή false αν δεν έχει.
